About the Project:

The project "Interactive-map-web-app" is a capstone project by Team Ahmed for SS24. It is a web application featuring an interactive map created using Leaflet and data from OpenStreetMap. The map provides content related to a commemoration. The repository contains HTML, CSS, and JavaScript files necessary for the application's frontend. For more details, you can visit the GitHub repository.
